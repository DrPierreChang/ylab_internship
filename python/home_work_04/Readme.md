### Создание виртуального окружения

```
python3.9 -m venv venv
```

### Установка Django 

```
python -m pip install Django
```

### Создание проекта
```
django-admin startproject app

```
### Создание приложения cookbook в директории проекта app
```shell
python manage.py startapp cookbook
```
